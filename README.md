🤖 Интеграция ИИ на STM32H750 с TensorFlow Lite Micro
📌 О проекте
Данный проект демонстрирует полную интеграцию искусственного интеллекта на микроконтроллерном уровне. Система выполняет инференс нейронной сети непосредственно на STM32H750, получая видеопоток с камеры OV2640 и выводя результаты классификации на LCD дисплей в реальном времени.

Ключевая концепция: Весь цикл обработки данных — от захвата изображения до вывода результата нейросети — выполняется на одном микроконтроллере без использования внешних вычислителей (компьютера, облачных сервисов).

🧠 Реализация ИИ на микроконтроллере
1. TensorFlow Lite Micro — ядро системы
Проект использует TensorFlow Lite Micro — фреймворк от Google, оптимизированный для выполнения нейросетей на микроконтроллерах с ограниченными ресурсами.
